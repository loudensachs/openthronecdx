## Original repository content in this project

- All gameplay code in `src/`
- All map JSON in `src/shared/maps/`
- All gameplay text, room names, province names, and UI copy
- All branding and the `OpenThrone` identity
- All deployment automation in `scripts/deploy.sh`

## Reference material

- OpenFrontIO was inspected only for high-level architectural patterns such as shared simulation, authoritative multiplayer, and readable territory-control UX.
- No OpenFront code, branding, assets, text, or map data were copied into this repository.
